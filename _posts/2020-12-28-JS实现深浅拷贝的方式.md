### JS实现深浅拷贝的方式

> + 浅拷贝：B赋值A，A修改，B随之修改。
> + 深拷贝： B赋值A，A修改，B不随之修改。

#### 浅拷贝

##### 1、Object.assign

```js
var obj = {
    a: 1,
    b: 3
}
var obj2 = Object.assign(obj);
obj.a = 2;
console.log(obj2.a); //2
```

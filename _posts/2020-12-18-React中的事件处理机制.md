### React中的事件处理机制

#### 阻止默认事件

原始写法

```html
<button onclick='eventHandle'>
    Active lasers
</button>
```

react

```jsx
function ActionLink() {
  function handleClick(e) {
    e.preventDefault();
    console.log('The link was clicked.');
  }

  return (
    <button onClick={handleClick}>
      Click me
    </button>
  );
}
```
